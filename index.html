Rover Swarm Co-Localization

During the summer of 2022, I was part of a student team working on a school internship for Lunar Outpost Inc. Our objective was to develop a method to localize rovers in a monocular camera frame using computer vision and machine learning techniques. Our team tackled the challenge of determining the transformation matrix that represents the position, orientation, and scale of the rover in the camera frame. Using a combination of feature detection and recognition, linear algebra, machine larning and computer vision techniques, we were able to extract meaningful information from company provided datasets and achieve our project's main goal. Adhering to Agile and Scrum methodologies, our team created a functional prototype solution, to be later used in real-world scenarios with MAPP rovers.

Population, Temperature, Pollution Database

I analyzed U.S. pollution, population growth, and daily temperatures from 2015 to 2019 by creating a relational database using PostgresQL. I performed data wrangling and cleaning, modeled the database using ERD, and normalized it up to fifth normal form. Using Jupyter notebooks and psycopg2, I designed and populated the database on a school provided server. I then worked with my classmates to analyze the resulting database and gained valuable insights in environmental trends. This project showcases my ability to create and manage a database, as well as my skills in database modeling, normalization, and data analysis.

Digit Recognizer

I developed a Digit Recognizer using a combination of numpy, pandas, and tensorflow within Jupyter Notebook. The project encompassed various steps, including dimensionality reduction and data visualization, which allowed for a deeper understanding of the dataset. To enhance model performance, I performed normalization to ensure consistent data distribution and faster convergence, along with label encoding and feature extraction. The dataset was split into training and testing sets for cross-validation. I implemented two distinct models—a Convolutional Neural Network (CNN) and a Principal Component Analysis (PCA) model—to compare their performance and accuracy. In addition, I employed confusion matrices to effectively evaluate and contrast the results obtained from both models, providing valuable insights into their respective strengths and weaknesses.

Languages: Python
Libraries: Scikit, Numpy, pandas, Tensorflow

Selective Face Blurring

Me and three other teammates (Benjamin Carlson, Caroline Rippey,and Jacob Tanchak) developed a proof of concept computer vision application focused on privacy protection. Leveraging Python and the powerful OpenCV library, we designed a facial blurring system based on a random distinct discriminant feature – red shirts. Using YOLOv4's model from OpenCV's Deep Neural Networks (DNN) module, we successfully identified individuals in frames and enclosed them within bounding boxes. Employing OpenCV's built-in Non-Maximum Suppression, we effectively filtered out redundant bounding boxes. For precise face detection, we implemented the Single Shot Detector (SSD). The key challenge was identifying the discriminant feature – red shirts – which we tackled through thresholding and a specialized computer vision technique known as morphological dilation. By pairing faces and red shirts within the bounding boxes, we achieved seamless facial blurring in both images and videos. Our project showcases a practical and efficient solution for safeguarding privacy in visual content.

Languages: Python
Libraries: OpenCV, YOLOv4
